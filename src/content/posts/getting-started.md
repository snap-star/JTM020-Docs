# Welcome to JTM020-Docs

This documentation site is powered by **Astro** using the **Fuwari** template, with content managed through **Obsidian** and **VaultCMS**.

## How to Navigate

Use the **navbar** at the top to explore:
- **Home** — Browse all documentation posts
- **Archive** — View docs organized by date
- **About** — Learn more about this project

Use the **search** icon to quickly find any documentation topic.

## Content Categories

Documentation is organized into categories:
- **Getting Started** — Setup and onboarding guides
- **Reference** — API and technical reference
- **Tutorials** — Step-by-step walkthroughs
- **Architecture** — System design and decisions

## Contributing via Obsidian

This site uses VaultCMS with Obsidian as the authoring interface. To contribute:

1. Clone the repository: `git clone https://github.com/snap-star/JTM020-Docs`
2. Open `src/content` as an Obsidian vault
3. Write or edit Markdown files — the VaultCMS plugin auto-configures your vault
4. Commit and push your changes
5. The site deploys automatically

> **Tip:** Use the Obsidian command palette and search for `Astro Composer` to create a new doc with the correct frontmatter template.
