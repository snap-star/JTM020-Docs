# Instrument-by-Instrument Setup Guide
## 1. DRUM KITS 🥁
Goal: Wide but punchy, with natural room ambience
| Parameter            | Setting                                                   | Why                                                        |
| -------------------- | --------------------------------------------------------- | ---------------------------------------------------------- |
| **Position**         | Slightly back in the field (middle-bottom of half-circle) | Creates a "band performing in front of you" feel           |
| **Width**            | 120-150%                                                  | Wider than mono but not exaggerated                        |
| **Distance > Late**  | 30-40%                                                    | Subtle room reverb                                         |
| **Distance > Early** | 20-30%                                                    | First reflections for realism                              |
| **Reverb Mode**      | **Wood** or **Concrete**                                  | Wood for warmer indie vibe, Concrete for brighter alt-rock |
| **Tail**             | Short-Medium                                              | Don't wash out the transients                              |
| **Wet**              | 0-15%                                                     | Keep it subtle—drums need punch                            |
 
> Pro Tip: Leave the kick drum slightly more centered (lower width, ~100%) and push the overheads/room mics wider. If you have separate snare top/bottom tracks, pan them slightly apart (10-15% L/R) for natural width.

## 2. BASS 🎸
Goal: Centered, solid foundation—resist the urge to widen this!
| Parameter | Setting | Why |
| --- | --- | --- |
| **Position** | Center, closest to listener (top of half-circle) | Bass must be mono-compatible |
| **Width** | 0-50% | Keep it narrow for punch and phase coherence |
| **Tilt** | Slightly down (negative) | Boost lows, cut highs for warmth |
| **Reverb** | OFF or 0% | Bass + reverb = mud in indie/alt rock |
| **Distance** | Minimal | Up front in the mix |
| **Pro Tip** | According to indie mixing techniques, use an aggressive low-pass around 140Hz if the bass interferes with the kick, and add harmonic distortion to help it cut through on smaller speakers . |

## 3. GUITARS 🎸
Goal: Wide, atmospheric, shimmering
| Parameter                | Setting                                 | Why                         |
| ------------------------ | --------------------------------------- | --------------------------- |
| **Rhythm Guitars**       | Hard pan L/R (far sides of half-circle) | Classic indie rock width    |
| **Lead/Texture Guitars** | Slightly back, 50-70% L/R               | Creates depth               |
| **Width**                | 150-200%                                | Make them expansive         |
| **Reverb Mode**          | **Cave** or **Heaven**                  | Atmospheric, washy          |
| **Tail**                 | Medium-Long                             | Let them ring out           |
| **Wet**                  | 20-35%                                  | Noticeable but not drowning |
| **Tilt**                 | Slightly up (positive)                  | Add air and shimmer         |

> Pro Tip: For a classic indie rock vibe, try hard-panning rhythm guitars to the far left and right (100% L/R) and placing lead or texture guitars slightly back in the field (30-50% L/R) with more reverb. This creates a lush, immersive soundscape without muddying the mix.  If you have a guitar with a lot of high-end sparkle, use the Tilt control to boost the highs slightly and make it shimmer in the mix. For darker, more brooding alt-rock tones, choose the Cave reverb mode and keep the wetness moderate to maintain clarity.

## 4. SYNTH & KEYBOARD 🎹
Goal: Atmospheric pads wide, arpeggiators focused
### For Synth Pads:
| Parameter       | Setting                         |                         |
| --------------- | ------------------------------- | ----------------------- |
| **Position**    | Back of field, wide L/R         |                         |
| **Width**       | 180-200%                        |                         |
| **Reverb Mode** | **Heaven** or **Dark**          |                         |
| **Tail**        | Long                            |                         |
| **Wet**         | 30-45%                          |                         |
| **LFO**         | Slow modulation on X/Y position | Creates subtle movement |

### For Arpeggiators:
| Parameter    | Setting            |
| ------------ | ------------------ |
| **Position** | Center to 30% L/R  |
| **Width**    | 100-130%           |
| **Reverb**   | Light plate reverb |
| **Wet**      | 10-20%             |

> Pro Tip: Use the LFO to modulate the X (pan) position slightly on pads—this creates a living, breathing texture. For arpeggiators, keep them more focused so they don't compete with the vocal .

## 5. VOCALS 🎤
Goal: Front and center, intimate but with dimension
| Parameter            | Setting                     | Why                                  |
| -------------------- | --------------------------- | ------------------------------------ |
| **Lead Vocal**       | Center, closest to listener | Must command attention               |
| **Width**            | 80-100%                     | Slight stereo enhancement, not wide  |
| **Tilt**             | Slightly up (positive)      | Air and presence                     |
| **Reverb Mode**      | **Wood** or **Concrete**    |                                      |
| **Tail**             | Short-Medium                | Intimate room, not cathedral         |
| **Pre-delay**        | 20-40ms                     | Keeps vocal clear before reverb hits |
| **Wet**              | 15-25%                      | Subtle space                         |
| **Distance > Early** | 10-20%                      | Slight room sense                    |

Backing Vocals:
| Parameter    | Setting                           |
| ------------ | --------------------------------- |
| **Position** | Wider (50-70% L/R), slightly back |
| **Width**    | 150-200%                          |
| **Reverb**   | Same as lead but 20% more wet     |
| **LFO**      | Optional slow pan modulation      |

> Pro Tip: In dense synth-heavy arrangements, consider using Panagement's delay instead of heavy reverb on vocals. A stereo slap delay (20-40ms L/R offset) can create width without muddying the mix .

- *Bass* for grit

# Advanced Immersive Techniques
## 1. The "3D Stage" Method
Arrange your mix like a real band on stage:
- *Drums*: Back center
- *Bass*: Front center
- *Guitars*: Back left/right
- *Vocal*: Front center
- *Synths*: Floating behind everything
Use the half-circle positioning to place each element at different depths .

## 2. LFO Movement for Texture
Set the LFO to:
- Waveform: Sine or Triangle
- Rate: 0.1-0.3 Hz (very slow)
- Target: X Position (pan) or Tilt
- Amount: Subtle (10-20%)
This creates evolving, living textures perfect for indie rock atmosphere.

## 3. Reverb Ducking
Use the LFO synced to tempo to duck the reverb wetness on the vocal—reverb pumps in between phrases but stays clear during lyrics.

:::tip
*Audacity Workflow Tips*
1. Apply Panagement 2 per track (not on the master) for individual control
2. Use the "Check Mono" button frequently—indie rock gets played on phones and Bluetooth speakers
3. Render tracks after setting spatial effects to save CPU (Tracks > Mix > Mix and Render) 
4. Save presets in Panagement for your "indie template"
:::

:::note
## Common Mistakes to Avoid
- Don't widen everything—bass, kick, and lead vocal should stay relatively centered
- Don't over-reverb—alternative rock needs clarity and punch; washy mixes sound amateur
- Don't ignore the Tilt control—use it to EQ elements back in the mix (cut highs = further away)
- Don't place everything at the same depth—use the distance control to create layers
:::
